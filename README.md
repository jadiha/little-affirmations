# little affirmations ✨

A cozy, cottagecore affirmation card website. Tap anywhere to draw a card — each one features a tiny animal character with a poetic, story-driven message for your day.

**[✦ Open the site](https://jadiha.github.io/little-affirmations/)**

---

## what it does

- 35 unique animal cards, each with its own illustrated character and handwritten-style affirmation
- Cards are shuffled and dealt without repeats — you'll see every card before the deck resets
- Glitter cursor trail on desktop, sparkle burst on mobile tap
- Subtle falling petals drifting in the background
- Card counter so you know how many you've drawn this session
- Fully mobile-friendly
